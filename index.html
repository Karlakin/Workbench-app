Harvest anywhere.

Your notes live in a private GitHub repo. This app reads and writes them through the GitHub API — every save is a commit. The token stays on this device only.

Token setup: github.com → Settings → Developer settings → Fine-grained tokens → scope it to only the notes repo, permission Contents: read & write, 90-day expiry. Nothing else.
